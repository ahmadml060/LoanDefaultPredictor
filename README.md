# Loan Default Predictor 🏦

Predicts loan defaults with 72.73% accuracy using a Gradient Boosting Machine.

### 🔗 Live Demo
Try the app here: **[Loan Default Predictor App](https://ahmadml060-loandefaultpredictor.streamlit.app)**

> Note: the app may take 30–60 seconds to wake up if it's been inactive.

### Key Metrics
- ROC-AUC: 0.7273
- F1 Score: 0.473
- Recall: 64.84%
- Features: 33 (15 raw + 12 engineered)
